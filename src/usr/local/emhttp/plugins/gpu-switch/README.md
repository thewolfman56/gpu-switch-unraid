# GPU Switch Manager

Intelligently manages GPU ownership between Docker containers and VMs.

## Features

- **Modern UI**: Drag-and-drop interface for GPU mapping
- **Real-time Monitoring**: Live GPU usage charts
- **Tdarr Integration**: Queue management before GPU switching
- **Multi-GPU Support**: Visual mapping for multiple GPUs
- **Automatic Switching**: Seamless transitions on VM start/stop

## Usage

1. Configure your VM and GPU mappings in the settings
2. Assign containers to GPU or CPU mode
3. Enable Tdarr integration if needed
4. The plugin automatically handles GPU switching
